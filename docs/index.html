What is zsh-aisuggestions?

A Zsh plugin that brings real AI understanding to your command line. Instead of naive history prefix matching, it uses LLMs to translate natural language, fix typos, complete arguments, and suggest context-aware commands based on your cwd, git state, last error, and recent history.

All API calls go directly to your configured provider. No intermediary servers. No data collection. No account required. Just your shell and your API key.

Features

Two modes, one goal — the right command, faster.

✏️ Ctrl+G

Rewrite Mode

Full command replacement. Translates natural language descriptions to shell commands, fixes typos, corrects flags, and rewrites based on context — last error, git state, cwd.

▶️ Ctrl+]

Autocomplete Mode

Inline ghost text at the cursor. Completes the current token, flag, or argument without replacing existing text. Like IDE autocomplete, but for the terminal.

🔌 Any API

Any LLM Provider

Works with any OpenAI-compatible endpoint: OpenAI, DeepSeek, Ollama, LM Studio, Groq, Together AI, and more. No provider-specific SDKs needed.

🐘 git clone

Smart Git Clone

Type git clone react and press Ctrl+G. The plugin queries the GitHub API to resolve the real repository URL — no LLM hallucination for repo paths.

📊 Context

Context-Aware

Sends cwd, git branch, dirty status, last exit code, last command, OS, and recent history to the LLM. Failed a build? It knows, and suggests accordingly.

🔒 Private

Privacy-First

All API calls go directly to your provider. No intermediary servers, no analytics, no telemetry, no account. Your data stays between you and your LLM.

Install zsh-aisuggestions

Two steps. Under a minute.

1

Run the installer

Via curl:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/EphemeralStudio/zsh-aisuggestions/main/install.sh)"

Or via wget:

sh -c "$(wget -qO- https://raw.githubusercontent.com/EphemeralStudio/zsh-aisuggestions/main/install.sh)"

Alternatively, from a local clone: git clone https://github.com/EphemeralStudio/zsh-aisuggestions.git && cd zsh-aisuggestions && bash install.sh

The installer automatically configures your ~/.zshrc — no manual editing needed.

2

Configure your LLM provider and restart

Edit ~/.config/zsh-aisuggestions/config.yaml to set your API endpoint and model:

api_base_url: https://api.openai.com/v1 model: gpt-4o-mini

Then set your API key using either option:

Option 1 — Directly in the config file:

api_key: sk-your-api-key-here

Option 2 — Via environment variable (recommended):

# In config.yaml: api_key_env: OPENAI_API_KEY # In ~/.zshrc or ~/.zshenv: export OPENAI_API_KEY="sk-..."

Then restart your shell:

exec zsh

Key Bindings

Minimal, intentional shortcuts. Nothing auto-triggers.

Key Action
Ctrl + G Rewrite mode — full command replacement, NL translation
Ctrl + ] Autocomplete mode — inline ghost text at cursor
Tab Accept suggestion (or normal tab-complete if none)
Right Arrow Accept suggestion
Backspace Dismiss suggestion and restore original input
